Sharing Content Using the Solstice App

Solstice improves meeting productivity and engagement with multi-participant content sharing, control, and markup. With the Solstice app on a laptop or mobile device, users can share and control content on the display connected to a Solstice Pod. (For installation information, see Getting the Solstice User App.)

Also see App-free Sharing to a Solstice Display to learn about using Windows Miracast or Apple AirPlay to mirror a device screen to the display without needing to download the Solstice app.
