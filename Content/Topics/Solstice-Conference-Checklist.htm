Solstice Conference Checklist

With Solstice Conference enabled on a Solstice Pod, Solstice becomes a BYOM video conferencing system that delivers rich content sharing to remote participants and supports touchless, agnostic conferencing for any meeting or learning space by providing wireless connectivity between user laptops and room audio/video devices.

Because Solstice does not burden a room design with single-purpose dedicated hardware, video conferencing with Solstice is far more versatile and cost effective. However, it does have some specific prerequisites to ensure a high-quality video conference experience. This document outlines those specifications and requirements of the various system components—host laptop, local area network, and audio/video peripherals—most of which are consistent with current best practice IT methodologies.

Host Laptop Requirements

  • macOS laptop that meets the following requirements:

  • Latest version of the Mersive Solstice app and Solstice Conference drivers matching the Solstice Pod installed on the meeting host laptop. Elevated permissions are required to install Solstice Conference drivers. If users do not have elevated permissions, administrators may centrally deploy the Mersive Solstice app using MSI or SCCMMSI or SCCM installers.
  • Screen resolution of 4K laptops scaled down to 1080p when connecting Solstice to a video conference.
  • No additional monitors attached to the host laptop during Solstice connections to video conferences.
  • Installed conferencing applications (Zoom, Teams, etc.) updated to the latest versions.

Local Area Network Requirements

  • Pod must be connected to the host network via Ethernet.
  • Connections between host laptop and Pod should have end-to-end latency of 50 ms or less.
  • Ensure your network has a minimum bandwidth of 50 Mbps per active Solstice Conference session/room.

    • Solstice Conference does not require any additional internet bandwidth beyond what is needed for your video conferencing services.

Solstice System Requirements

  • Gen3 Pod with current Solstice Subscription or Active Learning Subscription
  • Solstice Conference enabled using Solstice Cloud (or Solstice Dashboard)
  • The latest version of Solstice installed – see mersive.com/download for latest versions
  • Solstice Discovery Service (SDS) enabled (required for one-step meeting start)
  • Room calendar integrated with the Solstice Pod (strongly recommended for ideal user experience)

Audio/Video Peripheral Requirements

  • Compatible peripherals from the Supported PeripheralsSupported Peripherals list
  • Audio and video devices must be connected to Pod via USB
  • Peripheral power supplies should be used when available – Mersive does not recommend powering peripherals via the USB connection to the Pod

Limitations

  • Level of local area network bandwidth required is largely dictated by a combination of conferencing application and video camera used.

    Resolution: Use lower resolution cameras if local area network bandwidth is limited.

  • Audio crackling, degradation of streaming quality, and other performance issues may occur when the Pod is under high resource utilization.

    Resolution: Reduce number of shares, video content shared, and/or use lower resolution cameras if system performance issues occur.

  • PowerPoint should be started prior to initiating a Solstice Conference session if users want to share using PowerPoint’s presentation mode.

    Resolution: If PowerPoint is started after the Solstice Conference session begins, use PowerPoint’s option to "swap presenter view and slide show" under "display settings" when in presentation mode.

  • 4K host laptops are generally limited to a single desktop share; anything further may strain the laptop’s resources.

    Resolution: This issue will be resolved in an upcoming software release.

  • Meetings scheduled on a calendar integrated with the Mersive Solstice app do not appear in the Conference tab until 15 minutes prior to the meeting start time.

    Resolution: This is by design; Mersive will optimize product usability over time based on feedback from customers and partners.

  • The Mersive Solstice app supports integration with Microsoft 365 calendars only.

    Resolution: Addition of Google calendar support is on the product roadmap for the upcoming year. Other calendars will be added in the future.
