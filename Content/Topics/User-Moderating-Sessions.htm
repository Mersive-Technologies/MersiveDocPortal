Moderate a Solstice Session

Moderator mode allows you to approve requests from collaborators to join the session or post content to the display. Once moderator mode is enabled, any future collaborators will request to join and wait for a session moderator to approve the request. Only a moderator has the ability to control what content is posted and the layout of the content.
