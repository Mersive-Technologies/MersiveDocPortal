Solstice Cloud Analytics: Compare

The Compare analytics page in Solstice Cloud allows provides both graphical representations of and source data for usage data for specific options within a defined category. You can select to graph usage data for up to eight options within a category together and compare the same usage data for different defined options within a category. See Example.

This page is divided into two main parts:

  • Category Compare Graph - This is a graph at the top half of the page that displays data by category over a selected time period.
  • Displays Table - This table can list all the Pods that are being managed in your Solstice Cloud instance, or it can display a filtered subset of Pods. You can change options and apply filters that will change the data that appears in the table as well as apply to the data displayed in the table.

Insights from Analytics

Here are examples of how to generate graphs that can provide insights into how your Pods are being used.

How often are your meeting spaces used?

To determine how often meeting spaces that contain Mersive Pods are being used:

This example assumes you have defined a category named Room Type and that you have assigned Pods to options in that category. The category name used below is arbitrary; if you have a category that is roughly similar, choose the name of that category in step 4.

  1. In the left sidebar menu of the Solstice Cloud portal, go to Analytics > Compare.
  2. At the top right of the page, select the time period you want the graph: 48 hrs, 7 days, 30 days, or Custom. For information on specifying a custom time frame, see Set a Custom Time Period for Your Graph .
  3. In menu in the top right of the table, select the time increment you want for your graph: Days or Weeks.
  4. From the category drop-down, which is on the left just below the graph, choose Room Type.
  5. From the compare drop-down, choose Meetings.
  6. Using the check-boxes above the graph, select the options you want to display.

Each meeting space is rendered with a different colored line. By mousing over a plot point on the graph, you can display information about all the meetings spaces for that specific time period. Example

By selecting additional analytic data or categories in the Pods table, you can gain additional insights into the meeting context for each meeting. Example

How To

Below are the instructions on how to use the controls and filters on the Compare page.

Specify Time Increments for Your Graph Set a Custom Time Period for Your Graph
Select Category Options and Data to Compare
  1. In the left menu below the graph, select the category you want to graph.
  2. Select the Pod usage data you want to graph for the selected Pods from the right menu below the graph. This will be the vertical axis of the graph.
  3. Above the graph, check-boxes appear that list the available category options that you can display in the graph. Select the boxes for the options you want to compare. The color of the checkbox corresponds to the data line in the graph that shows the Pod usage for that category.

  4. You can mouse over graphed data points to see data details for that time period.
