Tested Configuration for Solstice + QSC Q-SYS

The following wiring diagram represents the room configuration that has been tested and verified by Mersive to be compatible with Solstice Conference. This document is intended to be used only as a reference and does not represent all possible supported configurations. For technical support with the QSC Q-SYS setup, please contact your local QSC representative. For technical support for Solstice Conference, please contact the Mersive support team.

Tested System Components

  • Solstice Pod Gen3 with Solstice Conference enabled
  • QSC Q-SYS Core 110f
  • QSC Q-SYS I/O USB bridge
  • QSC PTZ camera
  • QSC or third-party amplifier
  • QSC or third-party speaker
  • QSC supported third-party mic
  • PoE+ network switch
  • Display monitor

Dante microphones can also be used in this setup by plugging the microphones into the PoE+ network switch and configuring the Q-SYS Core 110f appropriately.

Additional Requirements

Meeting host laptop (Windows or macOS) with web conferencing app (Zoom, Teams, etc.) and Solstice app and bundled lightweight drivers installed
