Deploy the Solstice App with MSI or SCCM

For ease in centrally deploying the Solstice App, Mersive provides both MSI and SCCM installer packages of the Mersive Solstice app for Windows. MSI installers allows for a GUI-based installation on a local machine or GPO deployment in Active Directory, while the SCCM installer allows for a remote installation. Either of these methods can be used by your IT administrator to silently deploy the Mersive Solstice app and Mersive Solstice Conference drivers. Go to one of the topics below to learn more about how to centrally deploy Mersive user applications using one of these methods.
