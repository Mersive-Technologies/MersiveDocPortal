Step 4: Add a Room Calendar

Solstice offers options that allow you to display the schedule and calendar information for the room on a Solstice Pod display when there is no other content being shared. Participants can see if the space is currently scheduled or available, as well as the next three upcoming meetings in the space. For more information, see Solstice Cloud: Calendar Template.

Calendar Template

Use the following options to integrate room calendars with your Solstice Pod displays.

  1. In the left sidebar navigation panel of Solstice Cloud, click Manage > Templates > Calendar.
  2. Click Create New Calendar Template.
  3. Enter a descriptive Template Name.
  4. Select the Enable Calendar Feature checkbox.
  5. From the Calendar Type drop-down, select the type of calendar you are integrating for the room. You will need to provide the following information for each option:
    • Microsoft Exchange - Enter the Microsoft Exchange Server URL for the room calendar account, select whether your Exchange server uses Basic or NTLM as an Authentication Type, and enter the information needed for that authentication type as prompted. If the account uses either an Impersonation or Delegation mailbox, select the appropriate option.
    • Office 365 Online - Select the Authentication Type for the room calendar's O365 account and enter the information needed for that authentication type as prompted. If the account uses either an Impersonation or Delegation mailbox, select the appropriate option.
    • 3rd Party Only - Only select this option if you are using Solstice’s OpenControl API to integrate a third-party calendar. Configuration options are set through the OpenControl API.
    • Google Calendar - Upload service account credentials, such as a JSON key, for the Google Workspace service account associated with the Pod's room calendar.

    Authentication information for room calendar accounts, such as Username, Password, or Room Email, will need to be entered individually for each Pod. Steps for doing so are covered in Step 7: Apply Configuration Templates and Individual Settings.

  6. To hide meeting titles or meeting organizers from being visible on the room display, uncheck Show meeting titles and/or Show meeting organizers.
  7. Select the desired Update Interval frequency to set how often Pods will update the calendar meeting information visible on the display.
  8. Click Save.
Features Template

These settings determine which app-free sharing options users will be able to use to share content to the Solstice display. This template is also used to enable use of screen key for added security for Solstice collaboration sessions.

Additionally, screen mirroring for mobile devices is available through AirPlay and Android mirroring support. Miracast support for Windows devices can also be enabled. For more information on how to configure Solstice to support sharing with Airplay and Miracast, see Enable Sharing with AirPlay and Enable Sharing with Miracast.

  1. In the left sidebar navigation panel of Solstice Cloud, click Manage > Templates > Features.
  2. Click Create New Features Template.
  3. Enter the Template Name.
  4. Under the Solstice Feature Options section, select Enable screen key. This is an important security setting that requires end-users to enter the 4-digit screen key visible on the display to connect and share to Solstice.
  5. In the same section, enable or disable the various sharing app-free options based on your organization's needs and policies.
    • Miracast WiFi Direct: Allows users to mirror their Windows device screen; streams P2P from the Windows device to the Pod.
    • Miracast Infrastructure: Allows users to mirror their Windows device screen; streams over the existing network.
    • Android mirroring: Allows users to mirror their Android device screen in the Solstice app.
    • iOS mirroring (AirPlay): Allows users to mirror their iOS device screen.
      • AirPlay discovery proxy: Enable this option if your network does not allow use of Apple's Bonjour.
      • Bluetooth discovery for AirPlay: Provides another alternative for discovery for environments that do not allow UDP broadcast traffic or Apple's Bonjour protocol by allowing AirPlay users to discover the Pod using Bluetooth. Users will still have to connect to the same network as the Pod in order to stream content via AirPlay.
    • Clientless sharing via browser: Allows users to connect and share content via a web browser without the Solstice App.
  6. Click Save.
 Room Intelligence Template

These settings allow your Solstice configuration to sense and adapt to the room in which it is installed, integrating any conferencing solution installed on the host's laptop (Zoom, Teams, GoToMeeting, etc.) and automatically detecting useful information about a conference room.

  1. In the left sidebar navigation panel of Solstice Cloud, click Manage > Templates > Room Intelligence
  2. Click Create New Room Intelligence Template.
  3. Enter the Template Name.
  4. Select Enable Solstice Conference if you wish to enable the Solstice Conference capability.
  5. Click Save.
