Deploy the Solstice App with MSI or SCCM

For ease in centrally deploying the Solstice App, Mersive provides both MSI and SCCM installer packages of the Mersive Solstice app for Windows. MSI installers allow for a GUI-based installation on a local machine or GPO deployment in Active Directory, while the SCCM installer allows remote installation. Either method can be used by IT administrators to silently deploy the Mersive Solstice app and Mersive Solstice Conference drivers. Note that these admin-installed versions of the Mersive Solstice app and Solstice Conference drivers cannot be auto-updated by users upon connecting to a Pod running a newer version of Solstice.
