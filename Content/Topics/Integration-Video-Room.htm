Video Room Integration

Solstice Pods running Solstice 5.5 and higher can be configured to automatically switch between a dedicated Zoom Room, Microsoft Teams Room, or Cisco Webex Room and the Solstice Pod's multi-source wireless content sharing and agnostic video conferencing support. Pods can also be configured to support switching between Solstice and a supported all-in-one audio, video, and conferencing device.

System Components

Use the following components to integrate Solstice with a computer-based video room system:

  • Gen3 Solstice Pod running Solstice 5.5 or higher (licensing?)
  • Device running Zoom Room or Microsoft Teams Room software
  • Supported USB switcher that accepts RS-232 or REST API commands
  • Display monitor, video, and audio peripherals
  • Optional USB to RS-232 cord (for RS-232 USB switch)

When the video room system, display monitor, and USB audio and video peripherals (via a USB switching device) are connected to the Solstice Pod, Solstice 5.5 and higher can be configured to signal a USB switch with RS-232 or API commands to switch back and forth between and the video room system and Solstice's wireless sharing and agnostic video conferencing as needed. A second configuration is also possible to integrate Solstice with a supported all-in-one conferencing device that contains a camera, speakerphone, and conferencing software, such as the Poly Studio X30. In this type of integration, the supported all-in-one device is connected to the Solstice Pod via HDMI-In and USB. The Solstice Pod can then be configured to send API commands to switch the all-in-one device between its native functionality and an external device mode that passes AV input and output to the Solstice Pod.

How To Configure Solstice for Video Room Integration

Use the information below to configure a Solstice Pod for integration with a Microsoft Teams Room, Zoom Room, or Cisco Webex Room.

Attach Solstice Pod to Room System and Peripherals
  1. Attach the Solstice Pod to power and the local Ethernet and wireless networks, as appropriate for your Solstice deployment.
  2. The video room system (Zoom or Teams) device should also be powered on and connected to Ethernet on the same network as the Solstice Pod.
  3. Connect the display monitor to one of the Solstice Pod’s HDMI outputs (HDMI 1 or HDMI 2 on the back of the Pod).
  4. Connect the video room system device's HDMI output to the HDMI-In port on the front of the Solstice Pod.
  5. Connect any peripheral cameras, speakers, and/or microphones to the USB switch's USB output ports.
  6. Connect one of the USB ports on the back of the Solstice Pod to a USB input port on the USB switch.
  7. Connect a USB ports on the video room system device to a USB input port on the USB switch.
  8. Optional: If using an RS-232 USB switch, connect the other USB port on the back of the Solstice Pod to the RS-232 port on the USB switch using a USB to RS-232 cord.
 Configure Solstice for Dynamic Switching with Video Room System
  1. Open the Video Room Integration configuration page and enter login information if required.
  2. Check Enable Video Room Integration and select your Room System Type.
  3. Select the Device Switching Protocol based on the type of USB switch being used. If using an RS-232 switch, select RS-232.
    1. In Command to Select Solstice, enter the RS-232 command that corresponds with the port Solstice connects to on the USB switch.
    2. In Command to Select Room System, enter RS-232 command corresponding with the port the video room system connects to on the USB switch.
  4. If using a USB switch that accepts REST API commands, select HTTP/HTTPS.
    1. Under Command(s) to Select Solstice, enter an Order, Verb, Headers, Payload, and URL as needed to create an API command that will instruct the USB switch (or all-in-one device) to switch to the port connected to the Solstice Pod.
    2. Click Add HTTP Command to add rows for additional API commands as needed.
    3. Click the Delete (trash can) icon at the end of a command row to remove the associated API command.
    4. Under Command(s) to Select Room System use the tools described in steps a–c to create an API command that will signal the USB switch to switch to the port connected to the video room system device.
  5. Click Apply at the bottom of the configuration page to save your changes.
 Display a Custom Solstice Connection Message on Video Room System
  1. Open the Video Room Integration configuration page and enter login information if required.
  2. If Enable Video Room Integration is not selected, first complete the directions for configuring Solstice for video room system switching above.

  3. In Instructions Text Overlay section at the bottom of the page, enter the Solstice connection instructions you would like to appear to users on the video room system display. The variables [RoomName], [ScreenKey], [WifiNetworkName], [WifiIP], [EthernetName], and [EthIP] will automatically be replaced with specific details for that Solstice Pod.

    For example, entering the overlay text "To access Solstice, connect to [WiFiNetworkName], choose [RoomName], and enter screen key [ScreenKey]" would generate something similar to the following instructions:

    To access Solstice, connect to Company_Network, choose Conference Room, and enter screen key 4868

  4. Select a Text Overlay Position where you would like the Solstice connection instructions to be appear on the video room system screen.
  5. You may also choose a light or dark background from the Color Scheme menu.
  6. Click Apply at the bottom of the configuration page to save your changes.

Requirements & Recommendations

Follow these guidelines to achieve the best results with your Solstice video room integration:

  • Configure any devices to signaled by Solstice via API commands, such as the USB switch, to have a static IP on the same network as the Solstice Pod.
  • Set the USB switch to route to the Zoom Room or Microsoft Teams Room device by default.

Zoom Room Configuration

  • Disable device error notifications on the Zoom Room device (NEED MORE DETAILS HERE)
  • Set power mode for Zoom Room device to never sleep and never suspend display, except for times when the room won’t be used for any meetings, such as at night.

Microsoft Teams Room Configuration

  • Set power mode for Teams Room device to never sleep and never suspend display, except for times when the room won’t be used for any meetings, such as at night.
