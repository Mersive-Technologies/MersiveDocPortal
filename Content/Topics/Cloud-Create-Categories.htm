How To

Assign Automatic Categories

To utilize automatic categories, you will first need to enable Solstice Cloud to automatically set locations for the Solstice Pods in your deployment. Simply enable Location Services in the Pod’s Features settings. Location Services will use environmental scans from user devices that connect to the Pod to estimate the location of Pods using AI models that learn over time. After Pods have collected adequate data, Pod locations will be automatically set and the Pods will be assigned to automatic categories based on these locations (Country, City, and Building). These automatic locations can be viewed or updated on the Pod Location page (Manage > Location). For more detailed information, see Pod Location Settings.
