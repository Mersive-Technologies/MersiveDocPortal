Other Considerations

Below are some best practices that should be taken into account or performed after deploying Solstice.

  • If you want to learn more about how to enable and use Solstice Conference, see our Solstice Conference Guide. Solstice Conference Users Guide.
  • If you want to learn more about how to enable and use Solstice Active Learning, see our Solstice Active Learning Guide. Solstice Active Learning Guide.
  • Depending on your network security policy, you may need to export a list of the Solstice displays' MAC addresses and provide it to your network team so they can whitelist them on the network. You can export the list from the Solstice Dashboard. For more information, see Getting Started with Solstice Dashboard.
  • If you centrally manage PC-based client applications, Mersive provides MSI and SCCM installers for the Solstice app. For more information, seeDeploy Solstice with MSI or SCCMDeploy Solstice with MSI or SCCM.
